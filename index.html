Projects

Automated Aluminum Bar Inspection System – Selector 623

This project involved designing a system to measure the speed and length of aluminum bars in real time using ultrasonic sensors and control logic. Developed in assembly language for the Dragon12-Plus S12 board, it includes a full HMI with buttons, LEDs, LCD, and a 7-segment display.

The system classifies each bar based on length and activates a spray mechanism when the bar meets a configurable threshold. It operates in multiple modes (STOP, CONFIGURE, SELECT) and uses state machines for control and timing.

Skills developed: embedded systems, RTOS, low-level I/O, HMI development.

PCS Design (IEEE 802.3 Clause 36)

As part of the final project for the Digital Circuits II course, I designed a Physical Coding Sublayer (PCS) block for the 1000BASE-X Ethernet standard, based on IEEE 802.3 clause 36. The main datapath included essential components such as the 8b/10b encoder and decoder, synchronization, and state machines.

The design was implemented in Verilog and validated through a loopback test: data transmitted via TXD was encoded, passed through the PCS, and correctly received through RXD without data corruption. The testbench simulated the transmission of a full Ethernet frame.

I worked collaboratively using GitHub and followed a structured methodology, simulation with Icarus Verilog, waveform analysis in GTKwave, and automation with a Makefile.

Skills developed: Verilog HDL, digital design automation, protocol-based hardware design, testbench development, and simulation-based verification.

Linear Intensity Gradients

This project was developed for the IE-0449 Computer Vision course at the University of Costa Rica. It focuses on implementing gradient-based edge detection using the Sobel operator, both in offline image processing and real-time video analysis.

The project consists of two parts: a C-based desktop application that processes .bmp images and generates grayscale, gradient (X and Y), magnitude, and thresholded edge images; and a ROS node that processes live video from a USB camera, displaying all intermediate steps in real time using OpenCV.

Through this project, I strengthened my skills in C programming, image processing (Sobel filters, thresholding), file handling, and ROS integration with OpenCV for real-time computer vision applications.

Digital Mixer on FPGA Nexys A7

As part of the final project for the Digital Circuits Laboratory course (IE-0424), I designed and implemented a real-time digital audio mixer on the Nexys A7 FPGA. The system generates musical notes through a PWM-based audio output, controlled by the onboard switches of the FPGA board.

Each switch activates a different musical note, synthesized by generating a square wave at a specific frequency. The system was built entirely in Verilog, the mixer was connected to the audio output jack of the Nexys A7, allowing for direct speaker playback.

Skills developed: Verilog HDL, PWM signal generation, modular hardware design, clock division, memory-mapped I/O, real-time digital audio synthesis, FPGA prototyping.

Meteorological Station using Arduino

This project involved building a meteorological station using humidity, temperature, and luminance sensors (via a photoresistor). The data collected from the sensors was displayed both on additional hardware (LCD screen and LEDs) and through a graphical interface built with WxPython.

The sensor data was sent via the serial port to the computer, where it was shown on the graphical interface, which also generated real-time graphs. The project included designing the necessary circuit to create a fully functional weather station.

Skills developed: Arduino, sensor integration, hardware design, C++ programming, WxPython for graphical interfaces.

Labyrinth Quest

"Labyrinth Quest" is a maze game where players must navigate through different levels, collecting coins to increase their score and a key to complete each level. The game features 10 levels with varying environments and music.

The project utilizes Pygame for game development, Numpy for sprite animations, PIL for pixel detection to build the labyrinths, and Json for saving high scores.

Skills developed: game development, sprite animation, pixel detection, score tracking, Pygame, Numpy, PIL, Json.
